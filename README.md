# Md Hassan — ML Systems & Cybersecurity Portfolio

Welcome to my personal portfolio project. This repository contains the source code for my professional website, showcasing my research focus in cybersecurity, applied machine learning, and systems-level engineering.

## 🚀 Overview

This portfolio is designed to highlight my technical skills, national-level qualifications (GATE, UGC NET), and featured projects in threat detection, traffic analysis, and ML systems engineering.

## ✨ Features

- **Modern Responsive Design**: Fully optimized for desktop, tablet, and mobile devices.
- **Interactive UI**: Custom cursor effects, particle animations, and reveal-on-scroll functionality.
- **Project Showcase**: Detailed cards with modal views for research projects.
- **National Achievements**: Special sections for GATE and UGC NET qualifications.
- **Glassmorphism Theme**: A premium, dark-mode focused aesthetic with glass effects.
- **Performance Optimized**: Lightweight icons (Lucide) and optimized assets.

## 🛠️ Technologies Used

- **Frontend**: HTML5, CSS3 (Vanilla), JavaScript (Vanilla)
- **Icons**: [Lucide Icons](https://lucide.dev/)
- **Fonts**: Google Fonts (Inter, Fira Code)
- **Animation**: CSS Keyframes and Scroll Reveal logic
- **Other**: Particles.js (via canvas)

## 📁 Structure

```text
├── css/              # Stylesheets (layout, animations, responsive)
├── js/               # JavaScript logic (main interaction, particle engine)
├── assets/
│   ├── images/       # Project thumbnails and profile photo
│   └── cv/           # Downloadable CV
└── index.html        # Main entry point
```
